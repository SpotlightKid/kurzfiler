KurzFiler

Features

  • reads and writes all types of Kurzweil Sound Files (.krz;.k25;.k26). Files can be "downgraded" to older versions of the instrument, i.e. a file originally saved by a K2500 with KDFX can be made compatible to a K2000.
  • displays all Objects contained in the file
  • joins several files to one
  • can delete individual Objects in the file
  • lets you rename every Object
  • Wave Import
  • Windows .wav export
  • Velocity Split Keymap Generation
  • KCompact included (Multi-Root Sample Generation)
  • unlimited undo and redo
  • set the rootkey of a sample
  • generate simple programs directly from the sample pool
  • toolbar

Look at the example pages to see what KurzFiler can:
Tour1 How to import Wav-Files and make a program (example show Kurzfiler version 0.96)
Tour2 How to generate velocity split keymaps (example show Kurzfiler version 0.98)

The Java Runtime Environment is needed. Doubleclick the .jar-file to start the program. Some compression programs (for example WinRar) take over the .jar-extension and simply extract the .class files in KurzFiler.jar. If this happens to you, please reinstall java.
